# GoKrazy Kernel: NanoPi Zero2

Everything you need to be able to run GoKrazy on your [NanoPi Zero2](https://wiki.friendlyelec.com/wiki/index.php/NanoPi_Zero2).

Supports:

- Gigabit ethernet
- eMMC storage
- M.2 storage

## Cloning the kernel repository

This repository clocks in at over 3 GB of disk usage, so you might want to clone
it as a shallow clone:

```sh
git clone --depth=1 https://github.com/jphastings/gokrazy-nanopizero2-kernel
```
